# CptRaytracer Demo
## Work in Progress Raytracer in c++
  This is My Raytracer Project
  that Have Path tracing for Photorealistic purposes
## File Description
  ### Program
  #### 1. BMP.cxx
   Purpose : Just Fill Header Section of Bmp File.
  #### 2. ModelOpen.cxx
   Purpose : Contain a Triangle Information And Parse .obj and .mtl File
  #### 3. Vec3.cxx
   Purpose : Heart of Vector Calculation
  #### 4. Render.cxx
   Purpose : Main Program
  ### Other
  #### teapot.obj
   Standard Utah Teapot
  #### Primitive.obj & Primitive.mtl
   My Simple Scene that contain four Primitive on top of Plane
  #### F-14-Model.obj & F-14-Model.mtl
   My custom made F-14 tomcat See:
